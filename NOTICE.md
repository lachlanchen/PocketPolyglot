# Publication Notice

PocketPolyglot is a toolkit for creating bilingual and interlinear learning books. The repository may include small sample data and preview images to demonstrate the format.

Do not assume that every source text, translation, OCR output, generated JSON chunk, or full generated PDF in a local working copy is redistributable. Publish full books only when the original text and translation are public-domain, licensed, or otherwise cleared for redistribution.

The public GitHub export is intentionally limited to tooling, templates, references, sample data, and first-page preview assets.
