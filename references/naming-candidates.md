# PocketPolyglot Naming Notes

This project needs a name that signals small-format books, bilingual reading, and reusable language-learning tooling. The current preferred name is **PocketPolyglot**.

## Selected Name

**PocketPolyglot**

Tagline:

> Generate beautiful pocket-size interlinear books for language learning.

Longer description:

> PocketPolyglot turns bilingual texts into ruby, pinyin, grammar-colored, line-aligned pocket books.

Why it works:

- "Pocket" describes the A6 / small-reader PDF output.
- "Polyglot" makes the project larger than Chinese/Japanese while still fitting the current corpus.
- The name is concrete enough for GitHub, a website section, and generated book covers.

## Candidate Names

| Name | Strength | Caveat |
| --- | --- | --- |
| PocketPolyglot | Clear, memorable, directly tied to pocket books and language learning. | Slightly long, but easy to understand. |
| LinguaLeaf | Elegant and book-like; good for language notes. | Less explicit about pocket-size output. |
| TwinText | Strong signal for paired bilingual text. | Sounds more like a data format than a publishing tool. |
| RubyBridge | Good for furigana/ruby workflows. | Too focused on ruby; pinyin and other language pairs are less visible. |
| Interleaf | Nice book metaphor for interlinear text. | Existing word/name; may be harder to search. |
| ParallelPages | Very clear for bilingual editions. | Less distinctive. |
| BabelLeaf | Language-rich and literary. | "Babel" can imply confusion rather than clarity. |
| Glossline | Accurate for glossed line-by-line books. | More technical and less friendly. |
| ReadAlong | Friendly and learner-oriented. | Broad; not specific to generated books. |
| Linewise | Strong for aligned-line reading. | Understates the book-design and ruby/pinyin features. |

## Public Positioning

PocketPolyglot should be presented as a **toolkit and publishing workflow**, not a repository of raw copyrighted books. It can show first-page previews, sample data, TeX templates, JSON schemas, and reproducible scripts. Full source texts and generated full-book PDFs should only be published when rights permit.
