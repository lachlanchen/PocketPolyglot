# Sources

Put original scanned PDFs and other large source files here.

The source PDFs are intentionally ignored by Git; keep prepared text in `data/` and OCR output in `ocr/`.

Organization rule: put multiple versions of the same book in one title folder, for example `sources/少爷 - Botchan/` or `sources/金阁寺/`. Keep author albums, collected works, boxed sets, and secondary/reference books at the `sources/` root.

See `Japanese-Literature-Guide.md` for the author/title reference table and reading guide used to plan Japanese literature source pairs.
